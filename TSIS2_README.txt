TSIS 2: Judge Agent (AI Evals) — Almaty IT News Scraper

Inputs:
  prd.txt              — Product requirements (get_almaty_it_news)
  code_submission.py   — Intentionally flawed code (no timeout, no try/except, no docstring, etc.)

Agent logic:
  system_prompt.txt    — Judge persona and JSON schema
  judge.py             — Script: reads PRD + code, calls API, writes report
  judge.sh             — Bash wrapper (source .env; python judge.py)

Output:
  compliance_report.json — Generated by judge.py (sample included; overwrite by running)

How to run:
  1. Copy .env.example to .env and add your key:
       OPENAI_API_KEY=sk-...   (OpenAI, preferred if you use it)
     or
       GOOGLE_API_KEY=...      (Gemini)
  2. pip install -r requirements.txt
  3. python judge.py
     or: bash judge.sh

Result: compliance_report.json with compliance_score, status, audit_log, security_check.
