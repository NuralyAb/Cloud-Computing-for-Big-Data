# Школа иностранных языков "LinguaPro"

## О школе
LinguaPro — онлайн-школа иностранных языков. Работаем с 2019 года. Более 15 000 студентов прошли наши курсы. Обучение проходит на собственной платформе с видеозвонками, интерактивными упражнениями и AI-помощником для практики.

## Языки
Мы обучаем: английский, немецкий, французский, испанский, китайский, японский, корейский, итальянский, португальский, турецкий.

## Форматы обучения
1. Индивидуальные занятия (1-на-1 с преподавателем): 50 минут. Расписание гибкое, согласуете с преподавателем.
2. Групповые занятия (3–6 человек): 60 минут, 2 раза в неделю по фиксированному расписанию.
3. Разговорные клубы: 90 минут, группы 4–8 человек, только практика разговорной речи.
4. Интенсивы: ежедневные занятия по 90 минут в течение 2–4 недель.

## Уровни
- A0 (с нуля) — для полных начинающих
- A1 (Elementary) — базовый
- A2 (Pre-Intermediate) — ниже среднего
- B1 (Intermediate) — средний
- B2 (Upper-Intermediate) — выше среднего
- C1 (Advanced) — продвинутый
Перед началом обучения проводим бесплатный тест на определение уровня (15 минут онлайн).
